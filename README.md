# Spring AI Intelligent Agent

## 项目简介
本项目是一个基于Spring框架的AI智能体示例，演示如何与OpenAI的GPT-3.5模型进行交互。

## 结构
- `src/main/java/com/example/aia`: Java源代码
- `application.yml`: 配置文件
- `README.md`: 项目说明

## 依赖
- Spring Boot
- Spring AI

## 配置
请在`application.yml`中添加您的OpenAI API密钥。

## API范例
使用POST请求发送问题到 `/api/agent/chat` 接口。
